## 5. Search

어떤 데이터가 이미 존재하는지 검사할 때 사용하는 알고리즘.

### 배열을 탐색하는 방법

#### 1) 선형 탐색

- 모든 요소를 순서대로 확인하기
- 대상이 정렬되지 않았을 때 사용
- find, index, indexOf, includes 등의 방법
- O(n) 의 시간 복잡도

#### 2) 이진 탐색

- 남아있는 요소의 절반을 제거하면서 탐색하는 방법
  - 한 방향으로 계속 요소의 절반씩 제거해 나간다 (분할 정복 개념)
- 정렬된 데이터에서만 사용 가능
- 선형 탐색보다 훨씬 빠르고 효율적
- O(log n)의 시간 복잡도
